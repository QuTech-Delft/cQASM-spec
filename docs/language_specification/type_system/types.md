The table below lists the available variable types in cQASM.

| Type  | Keyword | Form                      | 
|:------|:--------|:--------------------------|
| Qubit | `qubit` | -                         | 
| Bit   | `bit`   | -                         | 

!!! note

    Type keywords are [reserved](../structure/reserved_keywords.md) and, therefore, cannot be used as [identifiers](../structure/identifiers.md).
