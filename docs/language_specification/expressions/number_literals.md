|:---------------|:--------------------------|:--------------------------------------------|
| Integer        | 64-bit signed             | `[0-9]+`                                    |
| Floating-point | IEEE 754 double precision | `[+-]?([0-9]*[.])?[0-9]+([eE][+-]?[0-9]+)?` |
