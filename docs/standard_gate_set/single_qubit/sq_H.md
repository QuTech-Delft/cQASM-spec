# Hadamard gate

| Name | Operator | Example statement |
|------|----------|-------------------|
| H    | $H$      | **`H q[0]`**      |

## Description

Hadamard gate

Clifford gate

## Representation

$$\begin{align}
H &= R_\mathbf{n}\left(\left[\frac{1}{\sqrt{2}}, 0, \frac{1}{\sqrt{2}}\right]^T, \pi, \frac{\pi}{2}\right) = e^{i\frac{\pi}{2}} \cdot e^{-i\frac{\pi}{2\sqrt{2}}\left(X + Z\right)} \\
\\
H &= \frac{1}{\sqrt{2}}\left(\begin{matrix}
1 & 1 \\
1 & -1 
\end{matrix}\right)
\end{align}$$
