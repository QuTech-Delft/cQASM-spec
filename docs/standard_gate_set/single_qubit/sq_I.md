# Identity gate

| Name | Operator | Example statement |
|------|----------|-------------------|
| I    | $I$      | **`I q[0]`**      |

## Description

Identity gate

Defined as a rotation of $0$ radians around the _z_-axis and a global phase of $0$.

Clifford gate

## Representation

$$\begin{align}
I &= R_\mathbf{n}\left(\left[0, 0, 1\right]^T, 0, 0\right) = I \\
\\
I &= \left(\begin{matrix}
1 & 0 \\
0 & 1 
\end{matrix}\right)
\end{align}$$
